# random-minecraft-skin-generator
generates minecraft skin randomly
## libraries
```
pip install Pillow
pip install random
pip install numpy
```
## parameters
```python
face_type = 2          # default (1 or 2)
face = True            # default
color = get_rand_rgb() # color of eyes. you can add color in rgb in here. example: color = (123, 123, 123)
```
